Open Yukkuri Editor は YMM4（ゆっくりMovieMaker4）互換の
オープンソース動画編集デスクトップアプリです。

📦 ダウンロード Portable .exe GitHub →

🎬 Open Yukkuri Editor とは

YMM4（ゆっくりMovieMaker4）のプロジェクト形式（.ymmp）と互換性を持ちながら、 モダンなWeb技術（Electron + React + TypeScript）で構築された、高速で拡張性の高い動画編集エディターです。

⚡ 高速なタイムライン描画（Canvas）
🔌 プラグインシステム搭載
🤖 AIによる動画編集（MCP対応）
🔄 AviUtlスクリプト互換

✨ 主な機能

🎞️

タイムライン編集

Canvasベースの高速描画。ドラッグ&ドロップ・キーフレーム・レイヤー管理をサポート。

🔌

プラグインシステム

YMM4互換のプラグインAPI。エフェクト・ツール・音声合成プラグインを自作可能。

🎤

VOICEVOX連携

VOICEVOX・COEIROINK等の音声合成エンジンと統合。ずんだもん・四国めたんに対応。

🎨

50+ エフェクト

色調補正・ぼかし・変形・トランジションなど豊富なエフェクトをWebGLで処理。

🤖

AI操作対応（MCP）

MCPプロトコル経由でAIが動画編集を実行。Claude等で「ゆっくり動画を作って」と指示するだけ。

🔄

AviUtl互換

AviUtlのスクリプト（.anm/.obj）を評価するインタープリターを内蔵。既存の資産を活用。

📄

YMM4互換

.ymmp形式で保存・読み込み対応。YMM4で作成したプロジェクトを開いて編集可能。

📦

ポータブル動作

インストール不要のポータブル版。USBメモリからそのまま起動できます。
